# Welcome to Coolcrete!

Coolcrete is a Minecraft mod that adds slippery concrete blocks to the game.

To run the mod locally, set up a fabric modding environment, then run the following:

```bash
./gradlew clean
./gradlew runDatagen
./gradlew runClient
```

To build the mod, run:

```bash
./gradlew clean
./gradlew runDatagen
./gradlew build
```
